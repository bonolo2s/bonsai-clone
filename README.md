# My-bonsai-clone

## Description

This project is a clone of an existing website, created purely for learning purposes.

## Technologies Used
This project is built with the following technologies:

- HTML/CSS: Used for structuring and styling the web pages.
- JavaScript: Used for adding interactivity to the web pages through DOM manipulation.

## Features
Here are the key features of this project:

1. **Linking Multiple Frontend Pages Together**: This feature allows seamless navigation between different parts of the website.
2. **Different Hover States on Navbar**: Enhances user experience by providing visual feedback when navigating through the navbar.
3. **DOM Manipulation**: JavaScript is used to dynamically modify the HTML and CSS of the web pages.
4. **Responsive Design**: The website layout adjusts according to the screen size and resolution.
